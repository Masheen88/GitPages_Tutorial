GitPages Tutorial with HTML and React

Be sure your project is pushed to a GitHub repository before following the tutorials below to post a Webpage on GitPages using HTML or React App.

GitPages using HTML

Publishing a HTML Project to GitPages

  • An index.html file is required to reside on the root of the repository.
  • Go to your GitHub repostiory ie. https://github.com/your_user/your_repo
  • Click on the Settings icon.
  • Click Pages on the left-hand menu.
  • Under Branch click on the dropdown menu and select main.
  • Click Save and wait for the page to refresh.
  • Navigate back to the Repostiory page. You may need to refresh a few times until you see the Environments tab appear on the right.
  • Click on github-pages when it appears.
  • Once the site is active you will see View Deployment. You can click on this button to see the live preview of your website.
  • You can copy you URL and distribute as needed. It will look something like: https://your_user.github.io/your_repo_name/

GitPages using React

Publishing a React App to GitPages

  • Ensure you have pushed React to your GitHub repository.
  • install gh-pages package by running the following command in your terminal: npm install gh-pages --save-dev
  • Add the following properties to your package.json file: "homepage": "https://your_user.github.io/your_repo_name/" "scripts": { "predeploy": "npm run build", "deploy": "gh-pages -d build" }
  • Run the following command in your terminal: npm run deploy
  • Navigate to your GitHub repository and click on the Settings icon.
  • Click on Pages on the left-hand menu.
  • Under Branch click on the dropdown menu and select gh-pages.
  • Click Save and wait for the page to refresh.
  • Navigate back to the Repostiory page. You may need to refresh a few times until you see the Environments tab appear on the right.
  • Click on github-pages when it appears.
  • Once the site is active you will see View Deployment. You can click on this button to see the live preview of your website.
  • You can copy you URL and distribute as needed. It will look something like: https://your_user.github.io/your_repo_name/index.html
